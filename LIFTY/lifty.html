Descripción de Lifty

Lifty es una aplicación móvil diseñada para promover la accesibilidad, ayudando a personas con diversidad funcional a encontrar lugares y servicios adaptados. La app ofrece información verificada sobre accesibilidad en espacios públicos y privados, con una interfaz intuitiva que prioriza la inclusión. Desarrollé Lifty combinando HTML, CSS y JavaScript para la landing page, y frameworks móviles para la app, integrando APIs de geolocalización y datos en tiempo real. El proyecto nació de la necesidad de crear soluciones digitales que empoderen a las personas y fomenten la igualdad.

Interactividad de Lifty

Prototipos interactivos en Figma Y Adobe XD que muestran la navegación dinámica, filtros de accesibilidad y funcionalidades clave de Lifty, listos para su desarrollo.

Video Promocional

Este video promocional destaca la misión de Lifty: facilitar la accesibilidad para todos. Muestra la interfaz intuitiva y las funcionalidades clave de la app, transmitiendo inclusión y empoderamiento. Web de Lifty

La landing page de Lifty, disponible en liftyhelp.github.io/lifty/, fue creada con HTML, CSS y JavaScript, ofreciendo un diseño responsivo y accesible que cumple con los estándares WCAG. La web promociona la app, destacando sus funcionalidades: búsqueda de lugares accesibles, personalización de perfiles y guías detalladas. Utilicé una paleta de colores suaves, tipografía clara y animaciones sutiles para mejorar la experiencia de usuario. El desarrollo incluyó prototipado en Figma, pruebas de accesibilidad y alojamiento en GitHub Pages para un despliegue eficiente.
